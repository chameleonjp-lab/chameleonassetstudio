FIRST, KNOW THE DIFFERENCE

まず、3種類の「変化」を分ける

見た目が動いても、作品データが変わっていない場合があります。逆に、画面では目立たなくてもゲーム用情報が変わる操作もあります。

1

表示だけが変わる

パン、ズーム、グリッド、選択範囲、再生確認など。作品データへ保存されません。

2

作品データが変わる

画像加工、位置、原点、判定、フレームなど。操作後にブラウザ内へ自動保存されます。

3

ファイルを書き出す

PNG、ZIP、.casprojなどを端末へダウンロードします。書き出しても編集中の作品は変わりません。

自動保存 通常編集は、操作後しばらくしてSafari内へ保存されます。
元に戻す 多くの編集は戻せますが、再読み込みや一部のプロジェクト操作で履歴は消えます。
復旧点 画像加工前に最大3件を残しますが、作成失敗の可能性があるためバックアップではありません。
.casproj プロジェクト全体を再編集用に保存します。Undo履歴と復旧点は含みません。
FEATURE CATALOG

ボタン名から機能を探す

カードを開くと、押す場所、操作、変化、保存、元に戻せるかを確認できます。

H

ホーム

プロジェクトを作成 新しい作品 Undo不可
どこを押す
ホーム → 「プロジェクト名」→「作成」
操作
あとで見つけやすい名前を入力して作成します。
何が変わる
空のプロジェクトが1件追加され、編集画面が開きます。
保存と戻し方
ブラウザ内へ保存されます。編集画面の「元に戻す」では作成を取り消せません。
.casprojを読み込む プロジェクト全体 元を変更しない
どこを押す
ホーム →「.casprojを読み込む」
操作
「ファイル」アプリなどに保管した.casprojを選びます。
何が変わる
読み込んだ内容が、元とは別IDの新しいプロジェクトとして追加されます。
保存と戻し方
読み込みは一括保存され、Undo対象外です。元ファイルと既存作品は変更しません。
プロジェクトを削除・復元 ごみ箱から復元可 完全削除は不可逆
どこを押す
ホームの作品一覧 →「削除」。戻すときは「ごみ箱」→「復元」
何が変わる
通常の削除では、ごみ箱へ移動します。画像と復旧点も一時的に残ります。
注意
ごみ箱は最大5件です。「完全に削除」と「ごみ箱を空にする」は取り消せません。
大切な作品は、削除前に.casprojも端末へ保存してください。

何が変わる

原点ゲーム情報Undo可
できること
ゲーム内で素材を置くときの基準点を決めます。
操作
基準にしたい場所をタップするか、現在の原点をドラッグします。
画像は同じ→配置の基準座標だけが変わる
出力と戻し方
asset.jsonとZIPへ反映されます。「元に戻す」で戻せます。

asset.jsonとZIPへ反映されます。「元に戻す」で戻せます。
P

プロパティ

アセット種別設定を切替Undo可
できること
キャラクター、背景、タイル、アイテムなど、素材の役割を設定します。
何が変わる
種別に応じた設定欄とゲーム用情報が変わります。画像そのものは変わりません。
注意
別種別へ変えても以前の種別用設定は内部に残り、元の種別へ戻すと再び表示されます。
独立コピー・左右反転コピー・削除別アセットUndo不可
コピー
選択中アセットを、元と連動しない別アセットとして追加します。左右反転コピーは見た目と対応座標を左右反転します。
削除
選択中アセットと関連画像・復旧点を直接削除します。プロジェクト用のごみ箱には入りません。
戻し方
これらは「元に戻す」の対象外です。削除前に.casprojを保存してください。
新規アセットを作成新しい素材Undo不可
操作
名前、種別、正方形・横長・縦長・自由サイズ、作成時のひな型を選びます。
何が変わる
現在のプロジェクトへ空のアセットが1件追加されます。
戻し方
作成はUndo対象外です。不要ならアセット削除になりますが、その削除もUndoできません。
画像・連番・Sheet・Tileset・Atlasを取り込む確定前プレビュー確定後Undo可
違い
「画像を追加」は別アセットを作り、「画像レイヤーを追加」は選択中アセットへ重ねる画像を追加します。
操作
ファイルを選び、保存前プレビューで件数・失われる内容・注意を確認してから確定します。
何が変わる
確定前は何も保存されません。確定すると対象全体を一括保存します。
戻し方
確定後は1回の「元に戻す」で今回の取り込み全体を戻せます。
復旧点画像加工前保証なし
できること
トリミング・消しゴム・色調整などの前の画像へ戻します。
保持数
アセットごとに最大3件です。古いものから消えます。
重要
復旧点の作成に失敗しても画像編集は続きます。.casprojにも入りません。バックアップとしては使わないでください。
Asset canvasサイズ枠と座標を変更Undo可
できること
アセット全体の枠を1〜4096pxの範囲で変更します。
操作
幅・高さと、古い枠を置く9つの基準位置を選び、プレビュー後に適用します。
何が変わる
枠と、レイヤー・フレーム・原点・アンカー・判定・パーツ基準点の座標が同じ量だけ移動します。
変わらないもの
画像のピクセル、大きさ、倍率、判定の大きさは変わりません。枠外のデータも自動削除しません。
レイヤー一覧表示と重なり順多くはUndo可
できること
画像やガイドを重ね、表示、名前、順番、位置、倍率、回転を変えます。
表示
OFFにすると画面と画像書き出しから隠れます。データ自体は残ります。
ロック
キャンバス上で未選択レイヤーを選びにくくする補助です。完全な編集禁止ではなく、一覧から選んだ後の数値変更や画像加工は可能です。
レイヤー削除後は「検査」の警告を確認してください。古いフレームやパーツの参照が自動で完全清掃されない場合があります。
整列・等間隔配置複数レイヤーUndo可
操作
レイヤー一覧で複数のチェックを付け、基準と並べ方を選びます。
何が変わる
選んだレイヤーの位置だけです。画像の中身、倍率、回転は変わりません。
戻し方
まとめて1回の「元に戻す」で戻せます。
画像編集編集用画像Undo可
含む機能
ブラシ設定、選択範囲、文字、透明縁検査、余白、画像リサイズ、背景透過、色調整、パレット、色置換、輪郭、左右反転。
何が変わる
主に選択レイヤーの編集用画像です。元画像は変更しません。
注意
「パレットを抽出」と「透明縁を検査」は読み取りだけです。「適用」「追加」「リサイズ」を押すと画像が変わります。
ゲーム情報・ゲーム属性画像外の情報Undo可
できること
原点、アンカー、当たり判定、scoreやrarityなどの任意情報を設定します。
何が変わる
ゲームで利用する座標や値だけです。PNG・WebPの見た目には反映されません。
出力
主にasset.jsonとZIPへ入ります。
パーツ・リグ動きの構造通常はUndo可
パーツ
複数レイヤーを頭・胴体・腕などの部位としてまとめ、回転の基準や親子関係を設定します。
リグ
時間ごとのパーツ位置・回転・倍率を作り、通常フレームへ焼き込みます。
焼き込み
フレームとアニメーションを追加します。後からリグを変えても自動更新されず、再度焼き込むと追加生成されます。
パーツ削除時に、子パーツやリグ内の古い参照が自動で完全清掃されない場合があります。削除後は検査結果を確認してください。
Family / Variant素材の派生関係多くはUndo不可
意味
同じ素材の元と、左右反転・色違い・手修正版の関係を記録します。これをFamily / Variantと呼びます。
自動更新
元を編集しただけでは派生版は変わりません。プレビュー後に明示更新します。
Undo
明示更新は戻せます。Family作成、登録、解除、Familyから外す、多くの派生版作成・削除はUndo対象外です。
出力
関係情報は.casprojへ入ります。通常のPNG、asset.json、ゲーム向けZIPには入りません。
Asset一括変更確定前プレビュー全体を1回でUndo
できること
最大16アセットへ、派生版更新、色置換、キャンバスサイズ変更をまとめて行います。
操作
対象を明示選択し、プレビューを準備して、実行可能・注意・変更なしを確認します。
保存
確定後は全対象をまとめて保存します。途中失敗では一部だけ残しません。
注意
保存開始後は中止できません。実行前に.casprojの退避を開けます。
T

タイムライン

フレームを追加・複製・削除表示状態を記録Undo可
フレーム追加
現在のレイヤー表示・位置・倍率・回転・透明度を1枚の状態として記録します。
削除
そのフレームと、各アニメーション内の同じフレーム参照を削除します。
前へ／後ろへ
フレーム一覧の並びだけを変えます。既存アニメーションの再生順は変わりません。
アニメーションを作成・再生再生順を保存再生は表示だけ
作成
名前を入れて作成すると、その時点のフレーム一覧順を再生順としてコピーします。
設定
1秒あたりの枚数とループを設定し、使うフレームを追加します。
再生・停止・先頭へ
確認用の表示だけです。再生したこと自体は保存されません。
O

書き出し

PNG / WebP選択アセットの画像作品は変更しない
PNG
透過対応の完成画像です。幅広い環境で使いやすい形式です。
WebP
容量を小さくしやすい完成画像です。利用先の対応を確認してください。
含まれないもの
原点・アンカー・判定の線、グリッド、現在のフレーム再生プレビューは入りません。
asset.json選択アセットの情報画像なし
できること
選択中アセットの原点・アンカー・判定・フレーム・アニメーションなどを保存します。
含まれないもの
画像ファイル本体は含みません。
ZIP選択アセット一式ゲーム向け
できること
選択中アセットの画像、フレーム用シート、ゲーム用情報、補助ファイルをまとめます。
範囲
プロジェクト全体のFamily関係や、編集用の全画像を保存する再編集ファイルではありません。
順序
画像シートはフレーム一覧順、アニメーションは各アニメーションに登録した順を使います。
.casprojプロジェクト全体再編集用
できること
現在のプロジェクトにある全アセット、全画像、Family関係を1ファイルへ保存します。
使い道
Safariのデータ消去への備え、別端末への受け渡し、後日の再編集です。
含まれないもの
Undo履歴、復旧点、選択中ツール、ズーム、グリッドなどの画面状態は入りません。
該当する機能が見つかりません。ボタンに表示されている短い名前で検索してみてください。
EXPORT COMPARISON

どのファイルに何が入る？

同じ作品でも、書き出す目的によって入る内容が違います。

形式 完成画像 ゲーム用情報 プロジェクト全体 再編集 主な用途
PNG / WebP 入る 入らない 入らない できない 完成画像として使う
asset.json 入らない 入る 選択アセットだけ 画像がない ゲームへ座標や動きを渡す
ZIP 入る 入る 選択アセットだけ 編集原本ではない ゲーム向け一式を渡す
.casproj 全画像が入る 入る 入る できる バックアップ・受け渡し

分からないボタンを見つけたら、名前で検索してください

編集画面の「？ 操作ガイド」からいつでもこのページを別タブで開けます。作品は開いたまま残ります。
